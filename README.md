☕ Coffee Sales Dashboard
📌 Project Overview

This project showcases an interactive Coffee Sales Dashboard built in Microsoft Excel to analyze sales performance and provide actionable business insights. The dashboard enables users to monitor key performance indicators (KPIs), identify sales trends, and make data-driven decisions through interactive visualizations.

🎯 Objectives
	•	Analyze overall sales performance.
	•	Monitor profit, revenue, and order volume.
	•	Identify top-performing coffee products.
	•	Compare sales across regions and customer segments.
	•	Track monthly sales trends.
	•	Enable dynamic filtering using Excel slicers.

🛠️ Tools & Skills Used
	•	Microsoft Excel
	•	Pivot Tables
	•	Pivot Charts
	•	Slicers
	•	Conditional Formatting
	•	Data Cleaning
	•	KPI Design
	•	Dashboard Development

📊 Key Performance Indicators (KPIs)
	•	Total Sales
	•	Total Profit
	•	Profit Margin
	•	Total Orders
	•	Quantity Sold
	•	Average Order Value

📈 Dashboard Features
	•	Interactive slicers for dynamic analysis.
	•	Sales trend analysis over time.
	•	Product performance comparison.
	•	Regional sales analysis.
	•	Profitability tracking.
	•	Clean and user-friendly dashboard layout.

💡 Business Insights
	•	Identifies the highest-performing coffee products.
	•	Highlights peak sales periods and seasonal trends.
	•	Reveals regions contributing the most revenue and profit.
	•	Supports inventory planning and strategic business decisions.
